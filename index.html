Встраивание компонента в страницу

Для того, чтобы была возможность создавать React компоненты, необходимо предварительно загрузить скрипты для работы с React. Далее(если это необходимо), можно добавить babel, который позволит создавать компоненты с помощью jsx синтаксиса.
После выполнения условий описанных выше, пишем скрипт для создания компонента. Если нужно встраивать компонент, по какому-либо событию, то создаём функцию в глобальном окружении и вызываем её, когда нужно создать
